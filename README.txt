#Site-gerador-de-senha

Nesse projeto coloquei em práticas meus conhecimentos em HTML, CSS e JavaScript.

Criei toda estrutura página com HTML e ja deixei tudo pronto para lincar com CSS e JS.

No CSS fiz toda estilização da página, deixando ela responsiva.

No JS adicionei toda funcionalidade da página, função no botão de gerar senha e copiar a senha gerada.
